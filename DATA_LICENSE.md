### Download the data

| Format | URL | Description |
|--------|-----|-------------|
| JSON (full graph) | `https://devlok-api.onrender.com/api/export/graph.json` | Nodes + edges in D3-compatible format |
| CSV (characters) | `https://devlok-api.onrender.com/api/export/characters.csv` | All characters as spreadsheet |
| CSV (relationships) | `https://devlok-api.onrender.com/api/export/relationships.csv` | All relationships as spreadsheet |
| Raw (seed file) | [`server/seed/seed.js`](./server/seed/seed.js) | The canonical source of truth |

### The philosophy

Sanatan Dharma's knowledge system has survived for 5,000 years because it was freely shared, debated, and built upon by communities across generations. A website cannot own that. This data was assembled from ancient texts in the public domain. We have structured it, written it, and verified it — but the underlying knowledge predates all of us by millennia.

**The dataset is our contribution to the commons. Keep it open.**
